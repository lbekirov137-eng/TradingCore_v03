# ORB Classic Baseline

## Назначение

Эта версия является эталоном.
Все будущие изменения сравниваются только с ней.

---

## Правила

1. Opening Range = первые 5 минут после открытия выбранной торговой сессии.

2. Вход только после закрытия свечи за диапазоном.

3. Обязательный ретест.

4. Подтверждение.

5. Стоп за диапазоном.

6. TP = 1:2 и 1:3.

7. Только одна открытая позиция.

8. Никакого усреднения.

9. Никакого входа без подтверждения.

10. Любое улучшение должно показывать лучшие результаты в Backtesting.
